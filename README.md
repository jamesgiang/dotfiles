### Depedencies 

ctags required for tagbar

`
brew install ctags
`

Cmake required for YouCompleteMe

`
brew install cmake
`

MacVim required for YouCompleteMe

`
brew install macvim --override-system-vim
`

Update bash_profile to include

`
alias vim='mvim -v'
`

### Installing new plugins

`
$ ./add.sh
`

### Updated submodules

`
$ ./update.sh
`
